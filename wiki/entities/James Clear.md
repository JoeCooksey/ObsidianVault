# James Clear

**Role**: Author, speaker, entrepreneur
**Known for**: *Atomic Habits* (2018); 3×/week newsletter at jamesclear.com (3M+ subscribers)
**Nationality**: American

## Key Contributions

- **Atomic Habits** (2018): synthesized decades of behavioral science into a practical, accessible framework for habit formation and change. Became one of the best-selling self-help books of the decade.
- Coined or popularized: **[[Two-Minute Rule]]**, **[[Habit Stacking]]** (formalizing Charles Duhigg's framework), **[[Goldilocks Rule]]**, **Never Miss Twice**, **Decisive Moments**
- Extended the [[Habit Loop]] from Duhigg's 3-step model (Cue → Routine → Reward) to a 4-step model adding the **Craving** phase, aligning more closely with neuroscience
- Formalized **[[Four Laws of Behavior Change]]** from the 4-step loop

## Core Ideas

| Idea | Summary |
|---|---|
| 1% Rule | Marginal gains compound: 1.01^365 ≈ 37.78× |
| Identity-Based Habits | Habits are votes for the person you want to become |
| Four Laws | Make it Obvious, Attractive, Easy, Satisfying |
| Never Miss Twice | Resilience rule: one miss is an accident, two is a pattern |
| Systems > Goals | Systems determine progress; goals only set direction |

## Intellectual Lineage

Clear synthesizes work from several researchers and predecessors:
- **Charles Duhigg** (*The Power of Habit*) — original Habit Loop (3-step)
- **BJ Fogg** (Stanford) — Tiny Habits; behavior design; motivation × ability × prompt
- **Nir Eyal** (*Hooked*) — product habit loops; similar craving mechanics
- **[[Wendy Wood]]** — habit and context dependency research
- **[[Andrew Huberman]]** — neuroscience underpinning (Limbic Friction, Task Bracketing) aligns with Clear's environmental design prescriptions
