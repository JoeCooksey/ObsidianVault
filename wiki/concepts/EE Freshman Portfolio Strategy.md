# EE Freshman Portfolio Strategy

Most EE students don't build a portfolio until junior year. A freshman who starts now arrives at sophomore internship applications with a visible edge.

## The Core Problem
Junior year is the "standard" time for EE internships. A freshman or sophomore applying early must compensate for lower coursework depth with **evidence of initiative** — which a portfolio provides.

## What to Build and Document

### GitHub Portfolio (Digital)
- Push every project, even small ones
- Write a README for each: what it does, what you used, what you learned
- Pin 3–5 best projects on your profile
- Include photos of physical builds if applicable

### Project Tiers by Year

**Year 1 (pre-EE courses):**
- LED blink / PWM on Arduino (demonstrates C code + GPIO)
- Voltage divider or RC filter on breadboard (photo + LTspice simulation screenshot)
- Python data plotter (simple CSV → matplotlib graph)

**Year 2 (EEE 120 + Circuits I era):**
- Simple FSM on Arduino (digital logic → code → hardware)
- Simulated filter in LTspice with measured vs. simulated comparison
- A small "system" project combining sensor + microcontroller + display

**Sophomore/Junior (targeting internships):**
- A complete schematic + PCB layout in KiCad for one of the above
- A power supply or motor driver circuit
- Anything WBG/SiC/GaN adjacent if targeting power electronics

## Portfolio Page Structure (per project)
1. **Title + dates**
2. **Objective** — what problem does this solve?
3. **Tools used** — hardware + software
4. **Photos** — fill ~1/3 of the page
5. **What you learned** — be specific; mention failures
6. **GitHub link**

## Key Insight
A freshman project does not need to be complex. A well-documented, clearly photographed Arduino voltage meter with a clean GitHub README beats a half-documented senior capstone every time. **Documentation is the differentiator.**

## ASU-Specific Resources
- **FURI** (Fulton Undergraduate Research Initiative) — apply sophomore year for research credit
- **ASU Solar Car Team** — hardware + embedded systems; open to freshmen
- **IEEE ASU Student Branch** — project workshops and networking
- **Makerspace at ASU** — laser cutters, 3D printers, soldering stations
