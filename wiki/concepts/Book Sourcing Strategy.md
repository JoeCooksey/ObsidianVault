# Book Sourcing Strategy

Framework for acquiring books at the lowest possible cost.

## Decision Tree

```
Want a book?
├── Is it a classic / public domain?
│   └── YES → Project Gutenberg (free)
├── Can you wait 1–2 weeks?
│   └── YES → Libby app (library, free)
├── Want the ebook cheap?
│   └── BookBub daily deal / Kindle sale
└── Want physical copy now?
    ├── Run BookFinder.com first
    └── ThriftBooks / AbeBooks / Better World Books ($2–6)
```

## Non-Negotiable Free Tools

- **Libby** — free ebooks + audiobooks from your library card; highest-leverage
- **BookBub** — free daily email of $0–$1 ebook deals; set genre preferences once
- **Project Gutenberg** — 70,000+ public domain classics

## Best Paid Sites

- **ThriftBooks** — best overall used book site; free shipping >$10
- **Better World Books** — cheap + social mission; global shipping
- **AbeBooks** — $1 books; good for obscure titles
- **Book Outlet** — 50–90% off new overstock
- **BookFinder.com** — compare all sites before buying

## Pricing Norms

| Condition | Typical Price |
|---|---|
| Public domain ebook | $0 |
| Library borrow | $0 |
| Used paperback (ThriftBooks) | $3–6 |
| Used hardcover | $4–9 |
| New overstock (Book Outlet) | $5–12 |
| New retail | $15–30 |
