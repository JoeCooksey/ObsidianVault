# Immersion Method Language Learning

The immersion method (popularized by **Refold.la** and the work of linguist Stephen Krashen) argues that language is *acquired*, not *learned* — meaning natural exposure to comprehensible input is more effective than drilling grammar rules.

## Core Theory

**Krashen's Input Hypothesis**: You acquire language when you understand messages in the target language slightly above your current level ("i+1"). Pushing output too early produces fossilized errors; comprehensible input builds the internal model first.

**Output Hypothesis (Swain)**: Production (speaking, writing) forces deeper processing and highlights gaps in knowledge. Both input and output are necessary, but in a specific sequence.

## Refold's Four-Stage Roadmap

### Stage 0 — Foundation (0–4 weeks)
- Learn phonetic system (IPA or target pronunciation guide)
- Install Anki; download a premade frequency deck (top 1,000–2,000 words)
- Pick a textbook for grammar reference (not drilling — just a map)
- Goal: understand the sound system; get ~500 core words in memory

### Stage 1 — Comprehension (months 1–6)
- Anki daily (5–10 new cards/day; never miss a review session)
- Begin passive immersion: TV shows, podcasts, YouTube with subtitles in target language
- Active immersion: rewatch the same content; look up unknown words in context
- Goal: understand ~50–70% of simple content (children's shows, basic podcasts)
- **Key insight**: Anki creates the "dictionary entry"; immersion fills it with real context

### Stage 2 — Intermediate Acquisition (months 6–18)
- Sentence mine: pull real sentences from your immersion content into Anki
- Increase immersion hours; switch to native content without learner subtitles
- Begin output: find a language exchange partner (iTalki, Tandem, HelloTalk)
- Shadow native audio to fix pronunciation before habits solidify
- Goal: hold a basic conversation; understand 70–85% of native content

### Stage 3 — Advanced Fluency (18 months+)
- Consume native media exclusively as entertainment (not study)
- Write daily (journal, Reddit, Discord servers in target language)
- Regular italki tutoring sessions for structured correction
- Goal: comfortable conversation on unfamiliar topics; near-native comprehension

## Spaced Repetition System (Anki)

Anki is the single highest-leverage tool in self-study language learning. How it works:
1. You see a flashcard
2. You rate recall (Again / Hard / Good / Easy)
3. The algorithm schedules next review — good recall → review in weeks; poor → review tomorrow
4. Result: you review every card at exactly the moment you're about to forget it

**Setup recommendations:**
- Deck: use a pre-made frequency deck first (top 2,000 most common words)
- New cards/day: 10–15 max (review pile compounds fast; don't overload)
- Time: 15–20 min/day for reviews is sustainable indefinitely
- Add context sentences, not just single words

## Critical Principles

**Consistency > Intensity**
15 minutes every day beats 2 hours once a week. Language acquisition is cumulative — the brain needs regular exposure to consolidate.

**Comprehensible Input Density**
Watching native content at 10% comprehension is nearly useless. Target 70–80% comprehension — that sweet spot drives acquisition.

**Track hours, not days**
Most learners reach conversational confidence around 200–300 hours of real exposure. Track cumulative hours to stay motivated.
