# Exercise Tier List — Back

Based on Jeff Nippard's evidence-based back exercise rankings (February 2024). Back training requires covering two planes: **vertical pull** (lat width — pull-ups, pulldowns) and **horizontal pull** (thickness — rows). Both are required for complete development.

---

## Back Anatomy for Programming

| Region | Function | Exercises |
|---|---|---|
| Lats (width) | Shoulder adduction/extension | Pulldowns, pull-ups |
| Mid traps / rhomboids (thickness) | Scapular retraction | Rows |
| Lower traps | Scapular depression | Face pulls, Y-raises |
| Rear delts | Horizontal abduction | Face pulls, reverse flyes |
| Spinal erectors | Extension | Deadlifts, hyperextensions |

---

## Tier List

### S-Tier — Gold Standard

| Exercise | Why |
|---|---|
| **Chest-Supported Row (machine or DB)** | No spinal loading; maximizes row ROM; best for mid-back thickness |
| **Wide-Grip Lat Pulldown** | Lat width; easy progressive overload; best for beginners to advanced |
| **Neutral Grip Lat Pulldown** | Longer ROM than wide-grip; slightly better for lats |
| **Cable Rows (various)** | Constant tension; both width and thickness; highly versatile |
| **Meadows Row** | Unilateral; maximizes lat stretch at the bottom; John Meadows' signature |

### A-Tier — Primary Lifts

| Exercise | Notes |
|---|---|
| **Wide-Grip Pull-Up** | Bodyweight king — challenging for beginners; excellent lat width |
| **Neutral Grip Pull-Up** | Easier than wide-grip; slightly better ROM for lats |
| **Deficit Pendlay Row** | Horizontal row from dead stop; excellent for strength + thickness |
| **One-Arm Dumbbell Row** | Unilateral with good stretch; easy to load heavy |
| **Kroc Row** | Heavy one-arm row with straps; extreme loading for thickness |
| **Dumbbell Pullover** | Targets lats in stretched position; also hits serratus |

### B-Tier — Good Secondary Options

| Exercise | Notes |
|---|---|
| Chin-Up | More bicep-dominant than pull-up; still excellent |
| Barbell Row | Classic thickness builder; technique-demanding |
| Pendlay Row | Explosive version of barbell row; good for power |
| Rope Face Pull | Critical for rear delts and external rotation health |

### C-Tier — Acceptable

| Exercise | Notes |
|---|---|
| Deadlift | Excellent overall exercise but not optimal for lat isolation |
| Yates Row | Underhand barbell row; lower back stress |
| Inverted Row | Good bodyweight option; limited loading |

### D-Tier — Suboptimal

| Exercise | Notes |
|---|---|
| Above-the-Knee Rack Pull | Trains nothing the deadlift doesn't; limited ROM |

### F-Tier — Avoid

| Exercise | Why to Avoid |
|---|---|
| **Renegade Row** | Requires so much core stability that back stimulus is minimal |
| Dumbbell hybrid row variations | Unstable setups that limit back loading |

---

## Programming Notes

- **Minimum viable back program**: S-tier row (chest-supported) + S-tier pulldown + face pull for rear delts/health
- Train at least 2 exercises per session — one vertical pull + one horizontal pull
- Face pulls are non-negotiable for shoulder health if you bench press
- Progressive overload via weight then volume — back responds extremely well to high volume (15–20 sets/week for advanced)
