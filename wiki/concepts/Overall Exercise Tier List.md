# Overall Exercise Tier List

Master ranking of all exercises across all muscle groups. Judged on: (1) hypertrophy stimulus per unit of fatigue, (2) progressive overload potential, (3) safety/injury risk, (4) stretch-based tension.

---

## S-Tier — The Best Exercises in the Gym

These exercises offer the highest ROI for muscle building across compound and isolation categories.

| Exercise | Primary Muscles | Why S-Tier |
|---|---|---|
| **Hack Squat (machine)** | Quads | Best quad isolation; deep stretch; no spinal load |
| **Romanian Deadlift (RDL)** | Hamstrings, glutes | Best posterior chain builder; maximum hip hinge stretch |
| **Machine Chest Press** | Chest | Best chest exercise; full ROM; safe; progressive overload-friendly |
| **Seated Cable Pec Flye** | Chest | Constant tension; excellent stretch; chest finisher |
| **Chest-Supported Row** | Back (thickness) | No spinal load; maximal row ROM; best mid-back |
| **Lat Pulldown (wide or neutral grip)** | Lats | Best lat builder; easy to load; beginner to advanced |
| **Cable Row** | Back (lats + mid) | Constant tension; both vertical and horizontal planes |
| **Cable Lateral Raise** | Lateral delts | Best side delt exercise; constant tension; no momentum |
| **Reverse Cable Crossover** | Rear delts | Best rear delt exercise; cross-body pull |
| **Machine Shoulder Press** | Front + side delts | Best overhead press for hypertrophy |
| **Overhead Cable Tricep Extension** | Triceps (long head) | Best tricep exercise; long head fully stretched |
| **Bayesian Cable Curl** | Biceps | Best bicep exercise; maximum stretch position |
| **Nordic Hamstring Curl** | Hamstrings | Highest eccentric load; injury prevention + hypertrophy |
| **Bulgarian Split Squat** | Quads, glutes | Best unilateral lower body exercise |
| **Barbell Back Squat** | Quads, glutes, core | King of compound lifts; strength foundation |

## A-Tier — Primary Training Movements

| Exercise | Primary Muscles | Notes |
|---|---|---|
| Barbell Deadlift | Posterior chain, traps | Overall king of strength; slightly less hypertrophy-specific |
| Incline DB Press | Upper chest | Better chest stretch than barbell |
| Barbell Bench Press | Chest, triceps | Strength staple; A not S due to limited stretch |
| Overhead Press (barbell) | Delts, triceps | Best compound shoulder strength |
| Weighted Pull-Up | Lats, biceps | Bodyweight king; scales with added weight |
| Meadows Row | Lats (unilateral) | Exceptional stretch; S-tier if you have the setup |
| EZ-Bar Curl | Biceps | Reduces wrist strain; volume workhorse |
| Preacher Curl (DB or machine) | Biceps | Deep stretch; excellent isolation |
| Barbell Skullcrusher | Triceps (long head) | Heavy stretch exercise |
| Close-Grip Bench Press | Triceps | Compound; all 3 tricep heads |
| 45-Degree Leg Press | Quads | Volume workhorse; safe; high rep capacity |
| Seated Leg Curl | Hamstrings | Hip-flexed position = better bicep femoris long head |
| Hip Thrust (machine or barbell) | Glutes | Best glute activation in the gym |
| Face Pull (rope) | Rear delts, external rotators | Critical for shoulder health |
| Incline DB Curl | Biceps | Extra stretch at bottom |

## B-Tier — Good Secondary Exercises

| Exercise | Primary Muscles | Notes |
|---|---|---|
| Barbell Row | Back | Classic; technique-demanding; good strength |
| Chin-Up | Lats, biceps | More bicep-dominant; complement to pull-ups |
| Leg Extension | Quads | Best pure quad isolation; excellent finisher |
| Goblet Squat | Quads | Beginner-friendly; limited ceiling |
| Dips | Chest, triceps | Deep stretch; shoulder health concern for some |
| DB Shoulder Press | Delts | Good free-weight press |
| Arnold Press | Front + side delts | Full ROM rotation |
| Hammer Curl | Brachialis | Arm thickness builder |
| Tricep Pressdown (bar) | Triceps lateral head | Easy to load; lateral head |
| Calf Raise (standing) | Gastrocnemius | Only calf exercise that matters |
| RDL to Lunge | Hamstrings + glutes | Combination movement |
| Rope Face Pull | Rear delt, external rotation | Same as A-tier for health benefits |

---

## C-Tier — Acceptable / Situational

| Exercise | Notes |
|---|---|
| Conventional Deadlift | Great for strength; not optimal for hypertrophy isolation |
| Inverted Row | Bodyweight; limited loading |
| Pushup | Beginner regression of bench press |
| Upright Row | Side delts; wrist stress risk |
| Leg Curl (lying) | Good; seated is slightly superior |
| DB Flye | Lower tension at top vs cables |
| Lunge | Good; balance-limited |

---

## D-Tier — Generally Avoid

| Exercise | Why |
|---|---|
| Front Raise | Anterior delts overtrained — redundant |
| Above-Knee Rack Pull | Cuts ROM of deadlift with no advantage |
| Bosu Ball Squat | Instability kills load and hypertrophy stimulus |
| Donkey Kicks | No resistance; activation drill at best |
| Bench Dips | Shoulder impingement risk; replace with dips or machine |
| DB Triceps Kickback | Light loads only; replace with cable kickback |

---

## F-Tier — Never Do These

| Exercise | Why |
|---|---|
| Guillotine Press | Bar lowered to throat; neck injury risk |
| Renegade Row | Core stability demand eliminates back stimulus |
| Jump Squat | Not hypertrophy; not beginner-appropriate |
| Behind-the-Neck Press | High impingement risk; no advantage over standard OHP |
| Smith Machine Good Morning | Spinal shear risk |
