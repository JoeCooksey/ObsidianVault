# Roblox AI Development Guide 2025 — Source Summary

A compiled source record for the autoresearch session on AI-assisted Roblox game development (2026-05-09). All URLs verified live during research.

## Primary Sources

### Official Roblox Sources

**Roblox Studio is Going Agentic (April 2026)**
- URL: https://about.roblox.com/newsroom/2026/04/roblox-studio-going-agentic
- Content: Official announcement of Planning Mode, agentic Building/Testing loops, parallel agent support. Data point: 44% of top 1,000 creators use AI tools. 50%+ of new Studio sessions from 2024–2025 joiners involved AI feature interaction.
- Relevance: Highest-authority source on current Studio AI capabilities

**Introducing Roblox Cube (March 2025)**
- URL: https://about.roblox.com/newsroom/2025/03/introducing-roblox-cube
- Content: Launch announcement for Cube 3D — Roblox's generative AI foundation model for 3D/4D content. Describes mesh generation beta, `GenerateModelAsync` Lua API.
- Relevance: Official source on AI asset generation capabilities

**Roblox Annual Economic Impact Report (September 2025)**
- URL: https://about.roblox.com/newsroom/2025/09/roblox-annual-economic-impact-report
- Content: Official 2025 creator earnings data. Platform-wide payout exceeded $1.5 billion. Top 1,000 creators averaged $1.3M; top 10 averaged $33.9M. Median developer: $1,440/year.
- Relevance: Ground truth for earnings data; cite this over secondary sources

**Roblox Creator Hub — Assistant Guide**
- URL: https://create.roblox.com/docs/assistant/guide
- Content: Official documentation for Studio Assistant features, capabilities, and usage
- Relevance: Reference for scripting integration details

**OpenGameEval Benchmark (December 2025)**
- URL: https://about.roblox.com/newsroom/2025/12/opengameeval-benchmark-agentic-ai-assistants-roblox-studio
- Content: Roblox's open benchmark for evaluating agentic AI assistants in game development context
- Relevance: Technical standard for comparing AI assistants
