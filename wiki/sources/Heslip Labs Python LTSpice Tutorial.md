# Cooking LTspice with Python — Heslip Labs

Best tutorial for getting started with PyLTSpice for simulation automation. Covers the most common automation patterns with complete working code.

## Automation Tasks Demonstrated

1. **Read simulation data** — extract traces from .raw files generated by LTSpice
2. **Compare circuits** — overlay multiple simulation results without manual work
3. **Run simulations** — automate schematic simulation via Python
4. **Parameter sweeping** — vary component values across multiple runs
5. **Model substitution** — step through different op-amp/transistor models
6. **Custom signal generation** — create PWL input signals programmatically
7. **RAW file writing** — merge results from multiple simulations
8. **Multi-simulation suites** — run all analysis types (.tran, .ac, .noise, .op) in sequence

## Libraries Used

- **PyLTSpice** — core simulation control
- **NumPy** — signal generation and numerical operations
- **Matplotlib** — result plotting
- **OS / Shutil** — file management

## EE Concepts Covered

RC transient response, AC frequency response, MFB filter design, op-amp noise, component tolerance effects

## Contribution to Research

Confirms all automation patterns in Projects 14–16 of the Phase 4 ladder. The multi-analysis suite pattern (Project 16 efficiency sweep) is directly applicable. Best reference for PyLTSpice beyond the official PyPI docs.
