# Research: Zero Cost Computer Skills You Can Build Today

**Query**: What skills can I build today, just in front of a computer, with no cost to entry?
**Rounds**: 3 | **Focus**: zero-cost, start-immediately, computer-only skill acquisition

---

## Key Findings

### Finding 1 — Touch Typing Is the Highest-Leverage Meta-Skill
Touch typing is the one skill that multiplies every other skill you build on a computer. The average hunt-and-peck typist does 30–40 WPM; a trained touch typist hits 70–100 WPM. At 8 hours of computer use per day, that is a ~30–50% throughput improvement on every other skill listed below. The best free trainers are **keybr.com** (adaptive, teaches key-by-key until mastery) and **typing.com** (structured curriculum from beginner to advanced). Most people reach functional touch typing (60+ WPM) in 4–6 weeks of 15 min/day practice. No account required to start on keybr.com.

### Finding 2 — Python Is the Highest-ROI Free Programming Language for 2026
Python is the dominant language of AI, data science, scientific computing, and test automation — and it is entirely free. The canonical free learning path: **"Automate the Boring Stuff with Python" by Al Sweigart** (full text free at automatetheboringstuff.com), then **freeCodeCamp Scientific Computing with Python** (free, no subscription), then the **Python EE Project Ladder** (already in wiki). Python's standard library + pip ecosystem mean 95% of useful tools are free. For an EE student, NumPy, SciPy, Matplotlib, and python-control are all `pip install` away. No purchase ever required.

### Finding 3 — Git and GitHub Are Free and Universally Expected
Git is the universal version control system used by every software, EE, and research team. GitHub free tier allows unlimited public repos and access to GitHub Skills (skills.github.com) — free interactive Git learning paths. A polished GitHub profile is worth more than a line on a resume for early-career engineers. The full "Pro Git" book is free online at git-scm.com/book. Starting from zero to a working first repo takes under one hour. This is non-negotiable for any engineer in 2026.

### Finding 4 — LTSpice Is a Professional EE Simulator With Zero Cost
LTSpice XVII is maintained by Analog Devices and is completely free — no trial, no subscription. It is used by professional EE teams worldwide for SPICE simulation of circuits. The 10-circuit ladder (voltage divider → RC filter → RLC → op-amp → buck converter) already documented in this wiki is executable the same day of installation. Combined with SparkFun's free written tutorial and Afrotechmods' free YouTube channel, a complete self-directed EE simulation education is free. This is the primary EE-specific zero-cost tool.

### Finding 5 — SQL Is a Durable High-Value Skill With Entirely Free Resources
SQL (Structured Query Language) is used in every industry for data querying and analysis. **SQLite** is the embedded database built into Python — requires no installation beyond Python. **Mode Analytics SQL Tutorial** (mode.com/sql-tutorial) is a free structured curriculum. **W3Schools SQL** and **Khan Academy SQL** are free. For engineers, SQL is used in automated test result databases, telemetry pipelines, and lab data logging. Skill acquisition to "useful in real projects" takes approximately 10–15 hours. Zero cost.

### Finding 6 — Verilog and Digital Logic Are Learnable In-Browser for Free
**EDA Playground** (edaplayground.com) is a browser-based Verilog/VHDL simulator. No installation required. Supports Icarus Verilog, GHDL, and commercial simulators via free account. This is the zero-cost on-ramp to FPGA engineering, which already appears in the wiki as a $175k average salary track. Free Verilog learning resources: HDLBits (hdlbits.01xz.net, entirely free Verilog practice problem set), Nandland (nandland.com, free FPGA/Verilog tutorials). Combined with EDA Playground, a student can write, simulate, and verify Verilog in-browser the same day.

### Finding 7 — Linux/CLI Proficiency Is Free via WSL on Windows
WSL2 (Windows Subsystem for Linux) is built into Windows 11 and free. Running `wsl --install` in PowerShell gives a full Ubuntu environment. Linux command-line proficiency is expected in every hardware lab, EV company, and national lab. The Linux Foundation offers "Introduction to Linux" on edX as a free-to-audit course. **The Missing Semester of Your CS Education** (MIT, free at missing.csail.mit.edu) covers the shell, editors, Git, and debugging in 11 lectures. This is a complete, free CLI education from MIT.

### Finding 8 — Prompt Engineering Is a Zero-Cost Skill With Immediate ROI
Effective use of Claude, ChatGPT, and Copilot is a genuine skill, not just typing queries. The gap between a novice and a skilled prompter is 5–10× in output quality. Anthropic publishes free prompt engineering documentation. OpenAI has a free Prompt Engineering Guide. The key techniques — chain-of-thought, role assignment, few-shot examples, output specification, iterative refinement — take a few hours to internalize and immediately amplify every other skill being built. Claude.ai has a free tier. No subscription required to learn this.
