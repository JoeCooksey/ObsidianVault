# PyPedia: Electrical Engineering with Python

GitHub repository of Python Jupyter notebooks covering EE fundamentals through worked examples.

## What It Contains

Three main sections:
- **01-Digital Circuits**: logic, Boolean algebra, gates
- **02-Circuit Theory**: resistive networks, nodal analysis, Wheatstone bridge, current dividers
- **03-Microelectronics**: semiconductor device models

## Libraries Used

NumPy, SciPy, SymPy, Matplotlib — standard scientific Python stack.

## Contribution to Research

Confirms the core library progression (NumPy → SciPy → SymPy) for EE circuit analysis. Circuit theory section validates the nodal analysis matrix approach in Projects 4–6 of the ladder.

## Limitations

No control systems or instrumentation content. Stops at circuit analysis level. Good reference for Phase 0–1 projects.
